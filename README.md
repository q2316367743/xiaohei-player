# 小黑影音

本地影音播放器

## 软件简介

小黑影音是一款功能强大的本地影音播放器，支持跨平台运行（macOS、Windows、Linux）。基于 Tauri 和 Vue 3 构建，提供流畅的用户体验和丰富的媒体管理功能。

### 主要特性

- **多格式视频播放**: 支持多种视频格式，包括 MP4、MKV、AVI、FLV、HLS、MPEG-TS 等
- **智能媒体库**: 自动扫描和解析本地视频文件，生成封面、预览视频和时间轴缩略图
- **元数据管理**: 支持编辑视频标题、描述、演员、标签、工作室等信息
- **播放历史**: 记录播放历史，支持断点续播
- **文件浏览**: 直观的文件浏览器，方便管理本地视频文件
- **NFO 文件支持**: 自动读取 NFO 文件获取视频元数据
- **跨平台**: 支持 macOS、Windows、Linux 三大桌面平台


## 安装注意

对于 `macos` 用户，如果出现文件已损坏，请在控制台执行`sudo xattr -r -d com.apple.quarantine /Applications/小黑影音.app`
